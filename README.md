# 📱 Canva OTP Bot for Termux

## 📌 Description
यह bot Termux में Canva accounts के लिए OTP send करता है।

## 🚀 Installation

```bash
# 1. Termux update करो
pkg update && pkg upgrade -y

# 2. Git install करो
pkg install git -y

# 3. Repo clone करो
git clone https://github.com/YOUR_USERNAME/canva-bot.git

# 4. Folder में जाओ
cd canva-bot

# 5. Requirements install करो
pip install -r requirements.txt

# 6. Chromium install करो
pkg install chromium -y
